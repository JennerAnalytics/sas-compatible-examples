# Παραδείγματα συμβατά με SAS (Jenner)

🌐 [English](../../README.md) · [Čeština](../cs/README.md) · [Dansk](../da/README.md) · [Deutsch](../de/README.md) · [Español](../es/README.md) · [Suomi](../fi/README.md) · [Français](../fr/README.md) · [Italiano](../it/README.md) · [日本語](../ja/README.md) · [한국어](../ko/README.md) · [Nederlands](../nl/README.md) · [Polski](../pl/README.md) · [Português](../pt/README.md) · [Svenska](../sv/README.md) · [中文](../zh/README.md)

Μια επιμελημένη, **επικυρωμένη** συλλογή αναλυτικών υποδειγματικών notebooks που
εκτελούνται στο [Jenner](https://jenneranalytics.com) — μια μηχανή DATA step και
PROC συμβατή με SAS.

Κάθε notebook που δημοσιεύεται εδώ έχει περάσει από **έλεγχο περιεχομένου ως προς
την αλήθεια των ισχυρισμών του**, όχι απλώς από επιβεβαίωση ότι εκτελείται:

- Εκτελείται χωρίς σφάλματα στον πυρήνα Jupyter του Jenner, και **οι έξοδοι της
  εκτέλεσης (πίνακες και γραφήματα) είναι ενσωματωμένες** — διαβάστε τα
  αποτελέσματα απευθείας στο GitHub χωρίς λήψη ή επανεκτέλεση.
- Κάθε αριθμός και κάθε συμπέρασμα της αφήγησης **θεμελιώνεται στην πραγματική
  έξοδο του notebook** — κανένα προδιατυπωμένο ή κατασκευασμένο εύρημα.
- Κάθε παραγόμενη εικόνα έχει **ελεγχθεί οπτικά**.
- Η διαδικασία SAS που αναγγέλλεται είναι αυτή που πράγματι εκτελείται.

Τα notebooks που δεν πληρούσαν αυτό το επίπεδο αποκλείστηκαν αντί να
δημοσιευτούν. Πρόκειται για ένα σκόπιμα μικρό σύνολο υψηλής αξιοπιστίας· μεγαλώνει
καθώς περισσότερα παραδείγματα περνούν τον έλεγχο.

## Notebooks στα ελληνικά

Αυτός ο κατάλογος περιέχει τις ελληνικές εκδόσεις των notebooks: έχουν μεταφραστεί
τόσο η αφήγηση σε markdown **όσο και** οι λέξεις-κλειδιά του κώδικα
(`ΔΕΔΟΜΕΝΑ`, `ΟΡΙΣΜΟΣ`, `ΕΚΤΕΛΕΣΗ`, …). Το Jenner εκτελεί αυτά τα προγράμματα
ως έχουν — η μηχανή αναγνωρίζει τη γλώσσα αυτόματα. Τα ονόματα μεταβλητών και οι
τιμές των δεδομένων παραμένουν αμετάβλητα, ώστε να ταιριάζουν με τις
ενσωματωμένες εξόδους.

- `procs/<διαδικασία>/<θέμα>/` — παραδείγματα οργανωμένα ανά διαδικασία SAS.
- `<κλάδος>/<θέμα>/` — παραδείγματα οργανωμένα ανά τομέα.

## Εκτέλεση παραδείγματος

```bash
jenner jupyter install
jupyter lab
```

Επιλέξτε τον πυρήνα **Jenner** και πατήστε «Run All». Επειδή οι έξοδοι είναι ήδη
ενσωματωμένες, μπορείτε επίσης απλώς να διαβάσετε κάθε notebook ως έχει.

## Μετάφραση προγραμμάτων

Για να μετατρέψετε οποιοδήποτε πρόγραμμα ή notebook μεταξύ γλωσσών — και προς
τις δύο κατευθύνσεις — χρησιμοποιήστε το
[`tools/translate.py`](../../tools/README.md):

```bash
python3 tools/translate.py --to el program.jenner   # αγγλικά → ελληνικά
python3 tools/translate.py --to en program.jenner   # ελληνικά → αγγλικά
```
